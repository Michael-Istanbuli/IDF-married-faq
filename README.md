# זכויות נשואים בצה״ל

מדריך קהילתי עצמאי בעברית לחיילים ולחיילות נשואים בשירות חובה: תשלומי משפחה, דיור, מענקים, חופשות, הורות וזכויות אזרחיות.

[למדריך החי](https://michael-istanbuli.github.io/IDF-married-faq/)

## עקרונות הפרויקט

- קובץ HTML יחיד, מהיר ועצמאי — ללא ספריות, גופנים, אנליטיקה או בקשות חיצוניות.
- מסלול קצר של ארבעה צעדים לאחר הנישואין ו־40 תשובות המחולקות לשישה תחומים.
- חיפוש מקומי בשפה חופשית, סינון לפי נושא, קישורים ישירים לתשובות, תרשימים ומחשבון המחשה.
- ממשק נגיש ומותאם לעברית ול־RTL, לטלפון ולמחשב, למצב כהה ולהעדפת תנועה או שקיפות מופחתת.
- מדיניות אבטחה מצמצמת: במדריך עצמו אין אחסון מקומי, עוגיות או איסוף מידע; קישורים חיצוניים כפופים למדיניות של אתר היעד.

## מקורות ואחריות

המידע מבוסס על פקודות צה״ל ומקורות ממשלתיים, ובמקומות המתאימים גם על הוראות אכ״א ומסמכי הקבוצה. כללים משפטיים נבדקים מול המקור הרשמי; סכומים מעשיים עשויים להסתמך על מסמכי הקבוצה ומסומנים בהתאם. הסכומים והכללים עשויים להשתנות. במקרה של ספק, הגורם הקובע הוא משרד הת״ש והמקור המחייב הוא נוסח הפקודה הרשמית.

המדריך אינו אתר רשמי של צה״ל ואינו מחליף בדיקת זכאות אישית.
