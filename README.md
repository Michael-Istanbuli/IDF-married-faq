# זכויות נשואים בצה״ל

מדריך שאלות ותשובות בעברית לחיילים ולחיילות נשואים בשירות חובה: תשלומי משפחה, דיור, מענקים, חופשות והטבות.

[למדריך החי](https://michael-istanbuli.github.io/IDF-married-faq/)

## עקרונות הפרויקט

- קובץ HTML יחיד, מהיר ועצמאי — ללא ספריות, גופנים, אנליטיקה או בקשות חיצוניות.
- חיפוש מקומי, סינון לפי קטגוריות, תרשימים ומחשבון המחשה.
- תמיכה מלאה בעברית וב־RTL, במצב כהה ובהעדפת תנועה מופחתת.
- מדיניות אבטחה מצמצמת: CSP, ללא אחסון מקומי וללא איסוף מידע.

## מקורות ואחריות

המידע מבוסס על פקודות צה״ל, הוראות אכ״א, מסמכי הקבוצה ומקורות ממשלתיים. סכומים וכללים עשויים להשתנות. במקרה של ספק, הגורם הקובע הוא משרד הת״ש והמקור המחייב הוא נוסח הפקודה הרשמית.
